Time Manager

Mobile application designed to improve time management and attendance tracking for on-site personnel.

It allows for schedule control, check-in and check-out logging, automatic report generation, and real-time performance monitoring—reducing administrative workload and improving operational efficiency.

✔ Identification of staff and administrative needs.

✔ Daily logging of attendance and working hours.

✔ Automated alerts and personalized notifications.

The app was deployed on Android devices used by on-site personnel. Training included in-person sessions and digital guides to ensure effective use. Improvements were implemented in short cycles, based on direct feedback from pilot users.

As a result, supervisors now receive automated reports with key attendance metrics and real-time alerts on schedule anomalies.

This development transformed time management into a digital, reliable, and automated process, facilitating real-time decision-making.

Control de Inventarios

The web application for inventory control was developed with the goal of centralizing and automating the management of equipment within the company. I used Google AppSheet tools for everything—from the database to the design and deployment of the web app—which enabled the creation of a fast, flexible solution accessible from any device.

One of the main challenges was the lack of control and standardization of elements within the construction industry, compounded by the constant turnover of personnel responsible for inventory management. Additionally, there was a widespread misconception about the role of the main warehouse: it was treated as a general storage facility, when its true purpose is focused exclusively on the storage and control of equipment.

Regarding the project's evolution, we began by collecting scattered information from various work sites, unifying criteria and formats to consolidate a reliable database. From there, the application grew in both functionality and visual design, incorporating user permissions, reports, and real-time traceability—enabling more precise, organized, and transparent inventory management today.

APU Generator

The application for generating Unit Price Analyses (APUs) was designed to streamline and simplify the creation process of these analyses—an essential tool in preparing budgets and proposals for construction project bids.

The goal was to develop a precise, intuitive, and efficient solution that allows users to calculate costs reliably and export results professionally.

  1. Development. Various technologies were used, including Unity for all development and web programming, SQL for database management and structure, and Figma for designing screens, buttons, and interaction levels.
  2. Project Challenges. The development faced several key challenges, such as identifying requirements through a detailed review of the company's current APU creation processes, and defining essential features like data input, automated calculations, and result exportation.
  3. Key Outcomes. Automated APU calculations based on real data, including both direct and indirect costs, and result export in formats like Excel, ready to be used in formal presentations or bidding processes.
